# SDLC — ペルソナ定義

---

## Persona 1: 佐藤 美咲（Misaki Sato）— 日本酒初心者

| 項目 | 内容 |
|---|---|
| 年齢 | 28歳 |
| 職業 | Webデザイナー（都内スタートアップ勤務） |
| ライフスタイル | 平日は残業多め、週末は友人とカフェや居酒屋。料理は自炊半分・外食半分 |
| 日本酒との関係 | 飲み会で出されれば飲むが、自分で選んだことはない。「辛口」「甘口」の違いも曖昧 |
| 技術リテラシー | 高い。スマホアプリを日常的に使い、新しいサービスへの抵抗感なし |
| 主な利用シーン | 居酒屋でメニューを見て迷った時、自宅で料理に合う日本酒を探す時 |
| ペインポイント | 1,400以上の蔵元・数千の銘柄から選べない。ラベルを見ても味の想像がつかない |
| ゴール | 自分の好みに合う日本酒を見つけ、少しずつ知識を広げたい |
| 言語 | 日本語 |

## Persona 2: 田中 健太（Kenta Tanaka）— 減酒志向エンジニア

| 項目 | 内容 |
|---|---|
| 年齢 | 32歳 |
| 職業 | バックエンドエンジニア（SaaS企業勤務） |
| ライフスタイル | 平日は朝会議が多く、翌朝のパフォーマンスを重視。週末は自宅で晩酌を楽しむ |
| 日本酒との関係 | 日本酒好きだが、飲みすぎて翌朝後悔した経験あり。最近は量をコントロールしたい |
| 技術リテラシー | 非常に高い。CI/CDの比喩がそのまま通じる。データドリブンな意思決定を好む |
| 主な利用シーン | 平日夜の「飲むか飲まないか」の判断、週末の晩酌の適量管理 |
| ペインポイント | 何杯が「ちょうど」かわからない。翌朝のパフォーマンスが賭けになる。「飲まない」と言いづらい空気 |
| ゴール | 飲む日も飲まない日も、データに基づいて最適な判断をしたい。翌朝を犠牲にしない |
| 言語 | 日本語 |

## Persona 3: Emily Chen — インバウンド観光客

| 項目 | 内容 |
|---|---|
| 年齢 | 35歳 |
| 職業 | マーケティングマネージャー（サンフランシスコ在住） |
| ライフスタイル | 年1〜2回の海外旅行が趣味。食文化への関心が高く、旅先ではローカルフードを積極的に体験 |
| 日本酒との関係 | ワインは詳しいが日本酒は未経験。旅行中に本場の日本酒を体験したい |
| 技術リテラシー | 高い。旅行中はGoogle Maps、翻訳アプリ、レストラン予約アプリを駆使 |
| 主な利用シーン | 日本旅行中のレストランで日本酒を選ぶ時、酒蔵見学の情報を探す時 |
| ペインポイント | 日本語のメニューが読めない。酒蔵の情報が日本語のみで辿り着けない。ワインの知識を日本酒に変換できない |
| ゴール | 日本酒の基礎を英語で学び、旅行中に自信を持って注文・体験したい |
| 言語 | 英語 |

## Persona 4: 山本 誠一（Seiichi Yamamoto）— 日本酒愛好家

| 項目 | 内容 |
|---|---|
| 年齢 | 45歳 |
| 職業 | 中小企業の経営者 |
| ライフスタイル | 週3〜4回の晩酌が日課。季節ごとに異なる銘柄を楽しむ。料理とのペアリングにこだわる |
| 日本酒との関係 | 20年以上の愛好歴。蔵元訪問も経験あり。製法や産地の違いを理解している |
| 技術リテラシー | 中程度。スマホは使うが、新しいアプリの導入には慎重 |
| 主な利用シーン | 自宅での晩酌時のペアリング探求、新しい銘柄の発見、味覚の記録と振り返り |
| ペインポイント | 好みが固定化しがち。新しい発見が減ってきた。飲酒量の自己管理が曖昧 |
| ゴール | 味覚プロファイルを可視化し、まだ出会っていない好みの銘柄を発見したい。ペアリングの精度を上げたい |
| 言語 | 日本語 |
