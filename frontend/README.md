# SDLC Frontend

React 19 + Vite + TypeScript による SPA。

## 開発サーバー起動

```bash
cd frontend
npm ci
npm run dev
```

http://localhost:5173 でアクセス。

## ビルド

```bash
npm run build    # dist/ に出力
npm run preview  # ビルド結果をプレビュー
```

## テスト

```bash
npm run test           # 全テスト実行
npm run test:coverage  # カバレッジ付き
```
